# Afondo - Higiene Industrial

Servicio premium de desengrase técnico y mantenimiento de sistemas de extracción en Alicante.

**Repositorio Oficial:** [https://github.com/jfrances12330/Afondo-Limpieza-](https://github.com/jfrances12330/Afondo-Limpieza-)

## Características principales
- Calculadora de presupuestos técnica en tiempo real.
- Generación de informes PDF certificados.
- Integración directa con WhatsApp Business.
- Diagnóstico asistido por IA Vision.

## Ejecución Local

1. Instalar dependencias: `npm install`
2. Configurar la API Key de Gemini en `.env.local`.
3. Ejecutar: `npm run dev`
